WPF 开箱即用项目模板

基于[HandyControl](https://github.com/HandyOrg/HandyControl)、[wpfui](https://github.com/lepoco/wpfui)项目搭建的WPF项目模板，用于项目的快速开发；



## Getting started
- [Visual Studio 2022](https://visualstudio.microsoft.com/vs/) with the following components:
    - .NET 6 SDK
    - .NET Desktop Develop
- F5 run
- Enjoy coding
